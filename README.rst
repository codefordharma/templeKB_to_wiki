=================================================
Generate wikipedia pages for temples of templeKB
================================================
* This repo is part of  IIIT-Hyderabad's Indian Language Wikipedia project.
* It uses Jinja2 template rendering to create wiki pages of templeKB temples. 
* It is modelled on Nikhil's repo for rendering karnatic raga pages in Indian Language Wikipedia project. (https://github.com/nikhilpriyatam/kb_to_wiki for more info.)
* For the temple corpus, please refer templeKB repo https://github.com/priyaradhakrishnan0/templeKB
================
Folder Structure
================
* corpus - contains 'WebTempleCorpus.json' from templeKB
* rendered_pages - contains indian-language folders holding the rendered pages
* templates - contains the indian-language templates
====================
Rendering articles
====================
python demo_wiki.py
